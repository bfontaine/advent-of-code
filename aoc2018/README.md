# aoc2018

My solutions to the [Advent of Code 2018][aoc18].

[aoc18]: https://adventofcode.com/2018

## Usage

```
lein repl
```
```clojure
(require '[aoc2018.p1 :as p1])

;; Problem 1.1
(p1/solution1 "input.txt")
;; Problem 1.2
(p1/solution2 "input.txt")
```

## Tests

```
lein test
```
